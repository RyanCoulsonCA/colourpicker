Setup

<link rel="stylesheet" href="colourpicker.css">
<script src="jquery.min.js"></script>
<script src="tinycolor.js"></script>
<script src="colourpicker.js"></script>

Include jQuery, TinyColor, and the ColourPicker files, and simply put an <input type="colour"/> wherever you need one, or initialise an input with $('#input').colourPicker(options).

Examples

Options

$('#input').colourPicker({
  autoSliders   : true,
  fallback      : 'black',
  fallbackAlpha : 'transparent',
  forceHex      : false,
  matchInput    : false,
  noAlpha       : false,
  useLastValid  : true,
});

Attributes

[placeholder="skyblue"]

The placeholder attribute is used as fallback when no input is given.

[no-alpha="true"]

Disables the alpha slider, disallows non-opaque colour. Same as noAlpha: true.

Methods

$('#input').colourPicker('val' [, string]);
$('#input').colourPicker('hue' [, 0-360]);
$('#input').colourPicker('saturation' [, 0-1]);
$('#input').colourPicker('value' [, 0-1]);
$('#input').colourPicker('alpha' [, 0-1]);
$('#input').colourPicker('show');
$('#input').colourPicker('hide');
$('#input').colourPicker('toggle');

Colour

get: $('input').colourPicker('val'); => 'blue'

set: $('input').colourPicker('val', 'red');

This is the current internal value of the ColourPicker, rather than what is in the input.

Hue

get: $('input').colourPicker('hue');

set: $('input').colourPicker('hue', 360);

Saturation

get: $('input').colourPicker('saturation');

set: $('input').colourPicker('saturation', 1);

Value (as in, HSV)

get: $('input').colourPicker('value');

set: $('input').colourPicker('value', 1);

Alpha

get: $('input').colourPicker('alpha');

set: $('input').colourPicker('alpha', 1);

Show Controls

$('input').colourPicker('show');

Hide Controls

$('input').colourPicker('hide');

Toggle Controls

$('input').colourPicker('toggle');
